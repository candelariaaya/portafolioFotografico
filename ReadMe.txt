Portafolio Fotográfico

La página web aquí creada fue desarrollada para utilizarse como portafolio online, pudiendo ser modificado para adaptarse a las necesidades de cualquier usuario.
Su interfaz es clara y sencilla, y posee un diseño web de tipo responsive para que pueda ser correctamente visualizada desde múltiples dispositivos.
Las imágenes que en ella se encuentran son propiedad exlusiva de María Candelaria Aya, sirviendo solo como ejemplo y no pudiendo ser empleadas fuera de este sitio.
